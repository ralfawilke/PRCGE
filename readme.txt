Code in relation to the research paper:

Lo, S.M.S., Wilke, R.A., Emura, T. (2023): A semiparametric model for the cause-specific hazard under risk proportionality, Working Paper.

The paper considers a semiparametric model for the cuase-specific hazard and a specification test. The following sample code is made available:

- "sample_code_PRCGE.R" contains sample R-code for the model. It can be used to replicate the results of the application. 

 "LFS.csv" contains the estimation sample.
